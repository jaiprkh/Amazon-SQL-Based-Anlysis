📊 Amazon India Data Analysis (SQL Project)
📌 Overview

This project focuses on analyzing an Amazon India dataset using SQL. It covers data cleaning, transformation, and analytical queries to extract meaningful business insights.

The project is divided into three main stages:

Data Cleaning
Data Analysis
Database Setup
🗂️ Project Structure
├── amazon_india_store_v3.sql   # Raw dataset / database schema
├── cleaning.sql               # Data cleaning queries
├── analysis.sql               # Analytical queries
└── README.md                  # Project documentation
⚙️ Technologies Used
SQL (MySQL / PostgreSQL compatible)
Database Management Systems (DBMS)
🧹 Data Cleaning

The cleaning.sql file includes:

Handling missing values
Removing duplicates
Standardizing column formats
Data transformation for analysis
📈 Data Analysis

The analysis.sql file contains queries to:

Analyze product categories
Identify top-performing products
Evaluate pricing trends
Generate business insights
🚀 How to Use
1. Setup Database

Import the dataset:

SOURCE amazon_india_store_v3.sql;
2. Run Data Cleaning
SOURCE cleaning.sql;
3. Run Analysis Queries
SOURCE analysis.sql;
